# Contributing to Covenant Weapon Security Protocol

Thank you for your interest in contributing to CWSP.

Whether you're reporting bugs, suggesting improvements, or contributing code, your help is greatly appreciated.

---

## Reporting Bugs

Before opening a new issue:

- Search existing issues first.
- Describe exactly what happened.
- Include steps to reproduce the problem.
- Include screenshots or videos if applicable.

---

## Feature Requests

Please describe:

- What you would like to see.
- Why it would improve CWSP.
- How it fits Halo gameplay.

---

## Code Style

- All code must be written in English.
- Function names must use the `CWSP_fnc_` prefix.
- Comments should explain **why**, not **what**.
- Keep functions focused on a single responsibility.

---

## Pull Requests

Please ensure:

- The project builds successfully.
- New features have been tested.
- Existing functionality is not broken.
- Update the CHANGELOG when appropriate.

---

Thank you for helping improve Covenant Weapon Security Protocol.

# Covenant Weapon Security Protocol (CWSP)

![Version](https://img.shields.io/badge/version-0.2.1-blue)
![Status](https://img.shields.io/badge/status-Alpha-orange)
![License](https://img.shields.io/badge/license-MIT-green)

> *"Unauthorized users are not permitted to operate Covenant weaponry."*

CWSP is an immersive, multiplayer-focused security system for **Arma 3** inspired by the **Halo** universe.

The project introduces biometric authentication and configurable failsafe mechanisms for Covenant weapons, preventing unauthorized personnel from operating alien technology safely.

Rather than simply removing weapons, CWSP aims to recreate the feeling that Covenant equipment is designed exclusively for Covenant species.

---

## Features

### Current
- Covenant weapon detection
- Multiplayer compatible
- Dedicated server support
- Configurable weapon blacklist
- Biometric authentication system

### Planned
- Biometric scan countdown
- Weapon failsafe activation
- Drop weapon to abort authentication
- Weapon-specific explosion effects

- Fuel Rod plasma detonation
- Halo-inspired warning interface
- Covenant warning sounds
- CBA Settings integration
- Mission maker API
- Full addon release

---

## Design Philosophy

CWSP is designed to feel like a natural extension of Halo lore.

Instead of punishing players instantly, the system creates tension.

Example:

```
COVENANT WEAPON SECURITY PROTOCOL

WARNING

FOREIGN BIOMETRIC SIGNATURE DETECTED

Authentication failed.

Failsafe activating...

3...
2...
1...
```

If the player drops the weapon before the countdown ends, the failsafe is cancelled.

Keeping the weapon results in catastrophic weapon failure.
